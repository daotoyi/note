# 摘录

  • 做事靠系统，不是靠感觉 P60

正如David Allen所说：‘我们应该在工作时尽量追求一种‘心静如水’的境界。空手道中用‘心静如水’来形容一切就绪的状态，我们可以想象把一粒石子投入沉寂的池塘中，池塘中的水会有何种反应呢？答案是：依照所投入物体的质量和力度做出相应的反应，然后又归于平静。池水既不会反应过激，也不会听之任之。 P67

  • 一次只处理一件事情：处理完一件事以后再去处理下一件。 P70

  • 压力是因为焦虑，焦虑是因为缺乏掌控感所造成的。 P88

  • 再微不足道的成就也要大肆庆祝！ P109

  • 时间管理管理的其实是承诺，履行承诺的能力就是时间管理的能力。 P114

  • 培养习惯的秘诀是少、慢，而不是多、快 P134

  • 培养习惯首先找到驱动力 P135
