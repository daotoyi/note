# greek

期权 (opens new window)的风险指标通常用希腊字母来表示，包括：  delta 值 (opens new window)、 gamma 值 (opens new window)、 theta 值 (opens new window)、 vega 值 (opens new window)、 rho 值 (opens new window)等

  • Delta （δ）=期权价格变化 / 期货价格变化
δ 值
价外期权 (opens new window) 平价期权 (opens new window)
看涨期权 (opens new window) 0 < δ < +0.5
看跌期权 (opens new window) -0.5 < δ < 0
  • Gamma（γ）= delta 的变化／期货价格的变化（表示Delta 对正股价格的变化最敏感）

与 delta 不同，无论看涨期权 (opens new window)或是看跌期权 (opens new window)的 gamma 值均为正值。

平值期权 (opens new window)的 Gamma 值最大，深实值 (opens new window)或深虚值 (opens new window)期权的 Gamma 值则趋近于 0。随着到期日的临近，平值期权 Gamma 值还会急剧增加。

当权证处于平价时，其 Gamma 值最大，这也意味着这时候 Delta 对正股价格的变化最敏感。

  • Theta（θ）  = 期权价格变化 / 到期时间变化 （此处时间是指距离到期日的时间长短）

Theta 的数值通常为负值，其绝对值会随时间消逝而变大，也就是说愈接近到期日，权证的时间价值消失的速度会愈快，最后到期时权证的时间价值应等于 0。

  • Vega（ν）= 期权价格变化 / 波动率的变化
  • Rho = 期权价格的变化／无风险利率 (opens new window)的变化

# expire

到期月份

合约的到期月份总是保持有四个月份，分别是当月、下月和随后的两个季月。所谓的季月就是指的 3 月、6 月、9 月和 12 月。如果目前是 3 月，那么 3 月就是当月合约，4 月就是次月合约，6 月就是下季月，9 月就是隔季月。

上证 50ETF 期权的到期日也就是最后交易日，是到期月份的第四个周三，如果是节假日则顺延到下一个交易日。

平价公式 ：认购期权价（C）+ 行权价（K） = 认沽期权价（P） + 股价（S） ^7fb259
